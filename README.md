# Attendance DApp (배포 후 15분 이내 정상 출석)

## DApp 개요
- **목표**: 라운드 시작 후 **15분(900초)** 동안만 `정시 출석` 허용, 이후에는 **지각/결석만** 허용.
- **검증**: 동료가 `verify(attendee)`로 당일 출석을 인정(최소 검증수 `requiredVerifiers`).
- **정산**: 라운드 종료 후 Top3에게 상금 분배(참가자 1~2명이어도 남는 몫 자동 재배치).
- **토큰**: 지각 시 `LateToken(LATE)` 1 토큰 민팅(메트릭용).


## 현재 상태(제출 메모)
- 프론트: **지갑 연결/주소 입력/출석·지각·결석·검증/기록·정산 UI 완성**.
- 컨트랙트: 요구사항 반영 완료(v1.1a, 남는 상금 처리 포함).
- 테스트넷 배포: **가스 부족으로 미실시**.
